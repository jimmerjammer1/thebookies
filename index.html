Connect Your Database

The Bookies needs a Firebase Realtime Database to sync data across all members in real time. This is free — setup takes about 3 minutes.

How to get your config:
1. Go to console.firebase.google.com
2. Create a project → Add a web app
3. Copy the values from the firebaseConfig object shown
4. In your project, go to Build → Realtime Database → Create database
5. Choose "Start in test mode" → paste your values below

Config is saved in your browser and never shared
